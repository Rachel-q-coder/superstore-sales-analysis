### 5. 技术亮点：窗口函数应用
使用RANK() OVER窗口函数进行品类内产品排名：

![窗口函数分析](result\data_rank-subproduct.png)

**SQL核心技术**：
```sql
RANK() OVER (
    PARTITION BY Category 
    ORDER BY SUM(Sales) DESC
) AS SalesRankInCategory
```

**业务价值**：
- 精准识别各品类明星产品
- 发现产品线优化机会
- 展示高级SQL技术能力
